Dental Clinic Automation System
About The Project
This is a comprehensive Dental Clinic Automation System built utilizing a microservices architecture. By separating the application into distinct, modular services (such as Business, Interface, and Service layers), the system ensures high scalability, easier maintenance, and robust performance for daily clinic management.

Key Features
Microservices Architecture: The system is built on loosely coupled services, allowing independent deployment and updates for business logic, UI forms, and data services.
Ready-to-Install Executable (.exe): A pre-compiled installer (DişKliniğiTakipKurulum.exe) is included in the repository, enabling users to install and run the application immediately without needing to compile the code.
Automated Setup Environment: Includes Inno Setup scripts (.iss) and the necessary SQL Server Express installer (SQL2022-SSEI-Expr (1).exe) to configure the database environment effortlessly.
Clinic Management: Streamlines patient records, appointment scheduling, and administrative workflows.
